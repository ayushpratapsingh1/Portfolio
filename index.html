About Me

Hello! I'm Ayush Pratap Singh, a passionate software developer specializing in Cloud Computing and DevOps. I enjoy designing scalable infrastructures, automating workflows, and deploying efficient applications on cloud platforms.

My DevOps journey includes working with tools like Docker, GitHub Actions, AWS (EC2, S3), and CI/CD pipelines. I’ve also set up production-ready deployments using Docker Compose and implemented secure pipelines with GitHub Secrets and EC2 instances.

On the web development side, I love building clean and responsive interfaces using React.js, Next.js, Tailwind CSS, and modern backend frameworks like Node.js and Express.js. My goal is always to deliver seamless user experiences through optimized frontend and backend integration.
